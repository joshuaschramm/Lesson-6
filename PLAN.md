# My Links - Project Brief
Personal link-in-bio page. One-page, clean design, mobile-first.
I want to send people here from my social profiles.

## Design
- Centered card no bigger than 480px wide
- Name in heading (Joshua Schramm), short bio below
- Profile Pic Placeholder (Circle 120px)
- Add smooth hover animations on link buttons
- Fonts - vintage inspired sans-serif
- backgroud - use Background.jpg in teh assets folder

### Visual Style
- **CMYK Offset Printing Effect**: Implement overlayed color separations with slight misalignment to create that authentic print registration look
- Cyan, magenta, yellow, and black layers should be visible and slightly offset from each other
- Apply this effect to key graphic elements (headings, profile pic border, or decorative elements)
- **Background**: Soft, blurry gradient (consider cyan to magenta or yellow to cyan transitions) to add depth without competing with foreground elements
- **Strong Contrast**: Bold black graphic elements paired with crisp white space
- High-contrast typography and shapes to evoke vintage letterpress/screen printing
- Consider halftone dot patterns or textures to enhance the print aesthetic
- Ink bleed or slight texture overlays for authenticity

### Color Palette
- Primary: Process Cyan (#00B5E2), Process Magenta (#E6007E), Process Yellow (#FFED00), Rich Black (#000000)
- Use pure white (#FFFFFF) for contrast and breathing room
- Colors should feel vibrant but slightly imperfect (like real ink on paper)
